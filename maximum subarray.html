class Solution:
    def maxSubArray(self, nums: List[int]) -> int:
        max_sum = nums[0]  # Initialize max_sum to the first element
        current_sum = nums[0]  # Initialize current_sum to the first element
        for num in nums[1:]:
            current_sum = max(num, current_sum + num)  # Update current_sum
            max_sum = max(max_sum, current_sum)  # Update max_sum
        return max_sum

The maxSubArray method takes in a list of integers nums and returns the maximum sum of any contiguous subarray within the given array.

We initialize max_sum and current_sum to the first element of nums.

We iterate through the remaining elements of nums starting from the second element.

For each element, we calculate the maximum sum of subarray ending at that element by comparing the element itself (num) and the sum of the previous subarray and the current element (current_sum + num).

We update current_sum with the maximum of these two values.

We update max_sum with the maximum of the current max_sum and the updated current_sum.

Finally, we return max_sum, which holds the maximum sum of any contiguous subarray within the given array.
