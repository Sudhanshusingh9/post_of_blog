class Solution:
    def myPow(self, x: float, n: int) -> float:
        if n < 0:
            x = 1 / x
            n = -n
        result = 1
        while n > 0:
            if n % 2 == 1:
                result *= x
            x *= x
            n //= 2
        return result

Explanation (Python):

We define a class Solution.

The myPow method takes in a floating-point number x and an integer n as input and returns the result of raising x to the power n.

If n is negative, we update x to be its reciprocal (1 / x) and make n positive by taking its absolute value (-n).

We initialize a variable result to 1 to keep track of the cumulative result.

We enter a while loop that continues as long as n is greater than 0.

Inside the loop, we check if n is odd by checking if n % 2 == 1.

If n is odd, we multiply the result by x.

We update x by multiplying it with itself (x *= x) and divide n by 2 (n //= 2) to reduce the problem size.

This process continues until n becomes 0.

Finally, we return the result, which holds the final result of x raised to the power n.
